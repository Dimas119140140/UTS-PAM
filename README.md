# UTS-PAM
1. Nama Aplikasi ini adalah "Cari kontak kita"
2. Deskripsi Aplikasi:
    Aplikasi ini berfungsi untuk mencari kontak yang tersedia pada perangkat ,dimana terdapat fitur search untuk mencari kontak,dan jika tidak ditemukan,maka akan muncul pemberitahuan "kontak tidak ditemukan".
3. Daftar package yang digunakan (expo):
1.contacts,untuk mendapatkan informasi mengenai contact
2.image manipulator untuk memodifikasi gambar tampilan yang disediakan
4. Cara menginstall aplikasi (dalam mode pengembangan)
    1.membuat folder untuk tugas.
    2.menginstal node js,instal react native cli,instal expo
    3.lalu membuat fila projek tugas tersebut.
    4.mengetik code.
    5.jalankan program dengan menggunakan aplikasi expo pada android,pada terminal jalankan npm start/expo start,lalu scan barcode .
